Introduction to ipcwqrPIC package

Introduction

ipcwqrPIC is the R package to fit a new inverse-probability censoring weighted (IPCW) estimating procedure for censored quantile regression when the data are partially interval-censored that include doubly-censored (DC) data and partly interval-censored (PIC) and possibly correlated within the same cluster. Let \(T\) and \(X\) be the event time of interest and its related \(p\)-vector of covariates, respectively. Our main objective is to estimate the \(p\)-dimensional quantile coefficient vector \({\boldsymbol{\beta}}_0(\tau)\) for some \(\tau \in[\tau_L,\tau_R]\subset (0, 1)\) in the following linear quantile regression model: \[ T_i = {\bf x}_i^T {\boldsymbol{\beta}}_0(\tau) + e_i(\tau),\quad i=1, \ldots ,n, \] where \(e_i(\tau)\) is the random error whose \(\tau\)th quantile conditional on \({\bf x}_i\) equals 0. When the data are subject to partially interval-censoring, left and right endpoints of the censoring time, \(L\) and \(R\), are observed instead of \(T\) such that \(T\in(L,R)\). Note that double-censoring can also be viewed as a special case of partly interval-censoring, i.e., \(T\) is left-censored if \(L=0\) and right-censored if \(R=\infty\).

Usages

Installation of ipcwQR package can be done by

devtools::install_github(repo="YejiStat/ipcwqrPIC")

or

base::require("ipcwqrPIC")

picrq() function has the following arguments:

picrq(L=L,R=R,delta=delta,x=x,tau=tau)

drq() function has the following arguments:

dcrq(L=L,R=R,T=T,delta=delta,x=x,tau=tau)

See the detailed description from help(picrq()) and help(dcrq()).

We first simulate univariate partly interval-censored (PIC) data with normal random error, which is similar to PIC data simulation setting of Kim et al. (2022+).

library(ipcwQR)
set.seed(111)
n = 200
x1 = runif(n,-1,1)
x2 = rbinom(n,1,0.43)
x = cbind(x1,x2)
T = 2 + x1 + x2 + rnorm(n)
U = (1 - 0.25*x1)*runif(n, -6, 5)
V = U + (1 - 0.1*x2)*runif(n, 6, 20)
U = exp(dplyr::case_when(TRUE ~ T, T>V ~ V, T<U ~ -Inf))
V = exp(dplyr::case_when(TRUE ~ T, T>V ~ Inf, T<U ~ U))
delta = ifelse(U==V, 1, 0)
tau=0.3
ipcwqrPIC::picrq(L=V,R=U,delta=delta,x=x,tau=tau)
#>           tau coefficients       se pvalue lower bd upper bd
#> Intercept 0.3     5.649208 0.154978      0 5.345451 5.952965
#> x1        0.3     5.656732 0.250324      0 5.166097 6.147368
#> x2        0.3     9.240053 0.249149      0 8.751721 9.728385
ipcwqrPIC::picrq(L=V,R=U,delta=delta,x=x,tau=tau,estimation = "dr")
#>           tau coefficients       se pvalue lower bd upper bd
#> Intercept 0.3     5.648980 0.154956      0 5.345265 5.952694
#> x1        0.3     5.656707 0.250266      0 5.166186 6.147228
#> x2        0.3     9.240259 0.249117      0 8.751989 9.728529

We posit two estimating methods, ipcw estimating method and doubly robust ipcw estimating method, which can be conducted by specifying estimation = NULL and estimation = “dr”, respectively.

Next, we simulate univariate doubly-censored (DC) data with normal random error, which is one of DC data simulation setting of Kim et al. (2022+).

n=200; tau=0.3
x1=runif(n,-1.2,1.7); x2=rbinom(n,1,0.6)
T = 1.7+x1+x2+rnorm(n)*(1-0.1*x2)
L=runif(n,-2.8,2); R=L+runif(n,4.2,8)
x=cbind(x1,x2)
delta=case_when(
    T<L ~ 3,
    T>R ~ 2,
    TRUE ~ 1 #observed
  )
ipcwqrPIC::dcrq(L,R,T,delta,x,tau)
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     1.186731 0.210573 0.0e+00 0.774008 1.599455
#> x1        0.3     0.983010 0.125846 0.0e+00 0.736352 1.229669
#> x2        0.3     1.059776 0.249825 1.1e-05 0.570119 1.549434
ipcwqrPIC::dcrq(L,R,T,delta,x,tau,estimation = "dr")
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     1.185831 0.210702 0.0e+00 0.772856 1.598807
#> x1        0.3     0.983015 0.125827 0.0e+00 0.736394 1.229637
#> x2        0.3     1.061370 0.249890 1.1e-05 0.571586 1.551153
ipcwqrPIC::dcrq(L,R,T,delta,x,tau,wttype = "nonparam", hlimit = 0.9)
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     1.186752 0.210666 0.0e+00 0.773846 1.599658
#> x1        0.3     0.983024 0.125819 0.0e+00 0.736419 1.229628
#> x2        0.3     1.059827 0.249902 1.1e-05 0.570019 1.549634

We can choose one of two weighting types, the parametric KM estimating method and Beran’s nonparametric KM estimating method, which can be accessible by using wttype=“param” and wttype=“nonparam”, respectively.

Finally, we cover multivariate doubly-censored (MVDC) data with normal random error, which is one of MVDC data simulation setting of Kim et al. (2022+).

cluster=60
v=rlnorm(cluster,mean=0,sd=0.5)
k = sample(0:9*10,cluster,replace = TRUE)
m = ifelse(quantile(v, k/100) <= v & quantile(v, (k+10)/100) <= v,
             k/10 + 2, 2)
id = as.numeric(unlist(apply(cbind(1:cluster,m),1,function(a) rep(a[1],a[2]))))
vi = as.numeric(unlist(apply(cbind(v,m),1,function(a) rep(a[1],a[2]))))
n=sum(m)
x1=runif(n,-1.4,0.9); x2=rbinom(n,1,0.56)
T = 2.3+x1+x2+rnorm(n)*x2*0.1+vi*rnorm(n)*0.85
L=runif(n,-2.8,2); R=L+runif(n,4.2,9.7)
x=cbind(x1,x2)
delta=case_when(
    T<L ~ 3,
    T>R ~ 2,
    TRUE ~ 1 #observed
  )
ipcwqrPIC::dcrq(L,R,T,delta,x,tau,id=NULL)
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     1.938231 0.199067 0.0e+00 1.548059 2.328402
#> x1        0.3     1.182625 0.187681 0.0e+00 0.814771 1.550480
#> x2        0.3     0.961907 0.254688 7.9e-05 0.462718 1.461096
ipcwqrPIC::dcrq(L,R,T,delta,x,tau,id=id)
#>           tau coefficients       se   pvalue lower bd upper bd
#> Intercept 0.3     1.921871 0.310542 0.000000 1.313209 2.530533
#> x1        0.3     1.120863 0.301445 0.000100 0.530031 1.711696
#> x2        0.3     0.934203 0.402944 0.010213 0.144434 1.723973

In real data analysis, we apply our proposed method to the multivariate clustered PIC data using a phase 3 metastatic colorectal cancer clinical trial. This dataset is accessible for data(mCRC) in the PICBayes R package (Pan, 2021).

library(PICBayes)
data("mCRC")
d = with(data.frame(mCRC), data.frame(U = ifelse(y==0,R,L),
                                      V = ifelse(y==2,L,R),
                                      # Cluster weighted data
                                      id=(rep(c(table(SITE)),c(table(SITE)))),
                                      # Treatment arm: 0 = FOLFIRI alone, 1 = Panitumumab + FOLFIRI.
                                      x1= case_when(TRT_C == 0 ~ 0, #Pan et al data
                                                    TRT_C == 1 ~ 1),
                                      # Tumor KRAS mutation status: 0 = wild-type, 1 = mutant.
                                      x2= case_when(KRAS_C == 0 ~ 1,
                                                    KRAS_C == 1 ~ 0),
                                      delta = case_when(IC == 0 ~ 1,
                                                        IC == 1 ~ 0)
));
L=d$U;R=d$V; delta=d$delta
L=(log(d$U));R=log(d$V); delta=d$delta
x = cbind(d$x1,d$x2); id=d$id;  tau=0.1;
ipcwqrPIC::picrq(L,R,delta,x=x,tau=tau)
